Clean Syntax

No semicolons, minimal punctuation. Code that reads like a well-formatted article.

Optional Types

Handle nulls safely with str? and the or operator for clean fallbacks.

Green Threads

Lightweight concurrency with go and channels. No callback hell.

Compiles to C

Generate portable C code. Easy interop with existing C libraries.

Why shotgun?

shotgun combines the best ideas from modern systems languages:

  • Odin's simplicity and data-oriented design
  • Zig's explicit memory control without the cryptic syntax
  • Python's clean aesthetics and readability
  • Rust's safety without borrow checker complexity

The result is a language where systems programming feels approachable. Write performant code without fighting your tools.

A Taste of the Language

Error Handling

fn read_config(str path) Config or Error {
    str contents = read_file(path) or error FileNotFound { path: path }
    Config cfg = parse(contents) or error ParseError { line: 10 }
    return cfg
}

Concurrency

fn main {
    chan int results = chan()

    go fetch_data(results)
    go fetch_data(results)

    int a = results.recv()
    int b = results.recv() or wait(100)

    print("got {a} and {b}")
}
