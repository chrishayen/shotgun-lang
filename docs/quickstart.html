Quick Start

Get up and running with shotgun in a few minutes.

Note: shotgun is in early development. There is no standard library yet. The compiler generates C code that you can link with your own runtime.

Prerequisites

The shotgun compiler is written in OCaml. You'll need:

  • opam - the OCaml package manager
  • OCaml 5.4.0 or later
  • A C compiler (gcc or clang)

Installation

Option 1: Download Binary (Linux only)

Download the latest release from GitHub Releases:

curl -LO https://github.com/chrishayen/shotgun-lang/releases/latest/download/shotgun-linux-x86_64
chmod +x shotgun-linux-x86_64
mv shotgun-linux-x86_64 ~/.local/bin/shotgun

Only Linux x86_64 binaries are available. macOS/Windows not supported.

Option 2: Build from Source

1. Install OCaml and opam

On macOS:

brew install opam
opam init
opam switch create 5.4.0

On Linux (Debian/Ubuntu):

apt install opam
opam init
opam switch create 5.4.0

2. Clone and build

git clone https://github.com/chrishayen/shotgun-lang.git
cd shotgun-lang
eval $(opam env)
cd compiler && opam install . --deps-only -y && dune build

3. Add to your PATH (optional)

alias shotgun="dune exec --root /path/to/shotgun/compiler -- shotgun"

Or copy the binary:

cp compiler/_build/default/bin/main.exe ~/.local/bin/shotgun

Hello, World

Create a file called hello.bs:

fn main {
    print("Hello, shotgun!")
}

Compile and run:

shotgun build hello.bs
./hello

You should see:

Hello, shotgun!

Compiler Commands

The shotgun compiler supports several commands:

# Compile to executable
shotgun build file.bs

# Compile with custom output name
shotgun build file.bs -o myprogram

# Output generated C code only
shotgun emit-c file.bs

# Type check without compiling
shotgun check file.bs

# Parse and show AST (debugging)
shotgun parse file.bs

Your First Program

Let's write something more interesting. Create greet.bs:

Person :: struct {
    name str
    age  int
}

Person :: greet(self) str {
    return "Hi, I'm {self.name} and I'm {self.age} years old."
}

fn main {
    Person alice = Person { name: "Alice", age: 30 }
    print(alice.greet())
}

Compile and run:

shotgun build greet.bs && ./greet

Output:

Hi, I'm Alice and I'm 30 years old.
